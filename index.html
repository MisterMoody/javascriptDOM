JavaScript and the DOM

The Document Object Model (DOM) is a programming interface for HTML and anything that is viewable in a web browser. The DOM is a data representation of all HTML loaded into the browser.

An Interface is an Object structured to enforce specific properties on an Object (or Class): Interfaces are designed as Objects, and ALL Objects are Properties of the Global Variable Window Interface, which represents a window containing a DOM Document and all of its Objects.

The creation of an Object begins by 'selecting' an <html> Element for a specified function referenced in the script. The reference is then used to manipulate it in the Document Interface, which is the Root Node of the HTML document itself. The Document serves as an entry point into the web page's content were Properties and Methods are used to control the structure, style, content and functionality of a Document on the web in a multitude of ways. These entry points are specific to the DOM Tree acting as Nodes that are accessible via Selectors aka <html> Elements.

In summary, working with the DOM requires controlling HTML Elements using JavaScript and this process consists of (a) selecting an element, (b) reading or modifying an element and (c) responding to user events.
