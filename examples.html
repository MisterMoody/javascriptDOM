Example 1: Modify Content using the HTMLElement.style Property

This example demonstrates how to modify content 'styles' of a targeted element that is bound to a 'click' event. The result is that the text color, background-color, border, font-size, font-weight, text-alignment and line-height styles for the element will change when the user clicks the element.

[b] Add Event Listener with Functions

This function starts by creating a variable that can reference the targeted elements. The document.getElementById() Method is one of several options that targets Elements for selection: this option targets the id 'textStyles' that is coded into the HTML <p> element.

Next, the addEventListener('event', () => { function}) Method is defined to use the variable reference 'textStyles' to listen for browser events. The first argument of this method is the 'type of event' that is expected: 'click' is the scheduled event that this listener is waiting for. The second argument of this method involves the functions that will be invoked by the listener whenever the user 'clicks' the element.

Inside of the listener, the variable name is referenced again along with the HTMLElement.style Property, which is used to specify a value to modify the elements content.

Example 2: Use a Button to Reset Modified Content

This example demonstrates how to reset modified content 'styles' of a targeted element that is bound to a 'click' event. All that was required was to include a <button> element with an id that could be used to target the 'textStylesZ' element: this is done by adding an event listener for the <button>. The difference between the two listeners is that the <button> listener targets the variable created to reference the <p> element.

The result is that the styles applied to the <p> on user-click will return to its original state.

A new addEventListener('event', () => { function}) Method is defined to use the variable reference 'resetButton' to listen for browser events on the <button>.

Inside of the listener, a 'click' event-type is passed as the first argument, but the second argument of this method allows events on the <button> element to occur on the <p> element.
