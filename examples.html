[b] Add Event Listener with Functions

This function starts by creating a variable that can reference the targeted elements. The document.getElementById() Method is one of several options that targets Elements for selection: this option targets the id 'textStyles' that is coded into the HTML <p> element.

Next, the addEventListener('event', () => { function}) Method is defined to use the variable reference 'textStyles' to listen for browser events. The first argument of this method is the 'type of event' that is expected: 'click' is the scheduled event that this listener is waiting for. The second argument of this method involves the functions that will be invoked by the listener whenever the user 'clicks' the element.

Inside of the listener, the variable name is referenced again along with the HTMLElement.style Property, which is used to specify a value to modify the elements content. In this example, the text color, background-color, border, font-weight, text-alignment, line-height and font-size are all being modified.
