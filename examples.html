Example 2: Use a Button to Reset Modified Content

This example demonstrates how to reset modified content 'styles' of a targeted element that is bound to a 'click' event. All that was required was to include a <button> element with an id that could be used to target the 'textStylesZ' element: this is done by adding an event listener for the <button>. The difference between the two listeners is that the <button> listener targets the variable created to reference the <p> element.

The result is that the styles applied to the <p> on user-click will return to its original state.

Example 3: Use an Input to Choose a Color

The example demonstrates how to change the color of an element by entering a color name or hex value into an input and using a button to confirm the event. An <input> of type 'text' is added to the previous code to designate an area to enter a color value.

The result is that the color of the headline text will change to whatever color value the user enters in the 'input' once the user 'clicks' the button.

DOM Traversal is a means of moving from one part of the DOM to another by selecting an element based on its relationship to another element. To do this, simply traverse to the parentNode using the parentNode Property. Take the elements parentNode and assign it to a variable parent, then use the parent to take an action: in this case, use the parent to remove an element (ie paragraph).
