[b] Add Event Listener with Functions

This function starts by selecting the element with the id 'textStyles' from the HTML <p> element then passing it into the document object using the 'getElementById' method.

Next, the 'addEventListener' method is defined to use the variable reference 'textStyles' to listen for browser events. The first argument of this method is the 'type of event' that is expected: 'click' is the scheduled event that this listener is waiting for. The second argument of this method involves the functions that will be invoked by the listener whenever the user 'clicks' the element.

Inside of the listener, the variable name is referenced again along with the style property followed by a CSS property that is equal to a specified value to modify the elements content. In this example, the text color, background-color, border, font-weight, text-alignment, line-height and font-size are all being modified.
